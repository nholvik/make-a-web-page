Lesson 1: The Basics of the Web & HTML

HTML or Hypertext Markup Language

HTML is made up of:

  • text content
  • Mark up
  • References
  • Links to to other pages

HTML is consisted of tags. You will typically have an opening tag and a closing tag

For example b tag is a bold tag.

HTML Attributes

Certain tags have attributes, you can use the A HREF tag used to hyperlink.

I can create a link by using the A HREF tag. For example, clicking this link will take you to google

You can also use the img tag to add an image to your webapage. To do this, you have to use the img src = "url" alt = "text" tag. For example, i'm adding this image of Hawaii by doing so:

When you use a img tag, it is a void tag because you do not need to use a closing tag.

Also, if the link is broken, the "text" part of the tag will appear. This is also useful for blind people.

Line Breaks

You can use the tag BR to create a line break

......This is also a void tag

Another tag you can use for a line break is the P tag, which is not void.

Inline/Block

Certain elements can be either inline or block. This means that they will either continue in the line you are working on, or they will create an invisible block around them.

br tag is used inline

p tag is block, it kind of creates an invisible box around the text where you can change the width and height

Some examples of inline elements

  • br
  • em
  • img

...the only block element we've learnt so far is P

Span/Div

  • Span - Inline
  • Div - Block

...these do nothing, but contain content

Lesson 3: Adding CSS Style to HTML Structure

Avoiding Repetition

Avoiding repetition is important so programmers can avoid errors. Consistency is also important to help avoid repetition. If you name each area that you want to look the same, the same div class, you only need to apply it once in the style sheet.

CSS

CSS stands for Cascading Style Sheets and it is used to give style to basic HTML documents. To do this, you can give a block of code a certain class, and then apply style to that class using CSS
